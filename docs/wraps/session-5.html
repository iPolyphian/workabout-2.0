⚖️ Decisions
Modal-based booking flow, not new route
Problem Booking could be a separate page or a modal overlay on the listing detail
Solution Modal keeps context (user sees the property behind). Booking state is ephemeral, not URL-synced, since it is a transactional flow.
Skip space selection when pre-selected
Problem Two entry points (property CTA vs space CTA) need different starting steps
Solution BookingModal accepts preSelectedSpaceId. If set, skip straight to date picker. Modal shows 3 or 4 steps accordingly.

Modal architecture pattern
